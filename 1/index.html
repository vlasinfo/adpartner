    3 Расположение виджета

    Укажите CSS селектор элемента на вашей странице, перед которым хотите разместить виджет

    ИЛИ

    Скопируйте якорный скрипт в html-документ вашего сайта, где хотите разместить виджет

    При первоначальной установке виджета происходит индексация страниц вашего сайта, которая может занять несколько часов в зависимости от размера сайта. Если по истечении этого времени виджет не появился, обратитесь в службу поддержки.
